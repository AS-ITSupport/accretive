The Calendar uses “Event Description” and “Calendar Event” content types.

  • The “Calendar Event” content type will hold the date and time.
  • The “Event Description” content type holds the link and description of each calendar event.
How to modify the date/time of the specific calendar event:
  1. Click on “Content Editing -> Content”. You will be taken to a page that lists all the contents of the site.
  2. Tick on “type” radio button and choose “Calendar Event” on the dropdown menu beside it, then click “Filter” button. It will now display all Calendar Event contents.
  3. Look for the event you wish to modify and click the “edit” link that is aligned to that particular event.
  4. The “Body” field can be left empty because this field will not be used anywhere.

    Repeating Events:
    The following are tips when adding events that repeat:

    • If the event repeats, you can set the “To date’ field either to an empty value or to the same value as the “From date” field.
    • In the Advanced>Repeat section, Choose a frequency and period to repeat the event. Set the ending event date on “Until” field (if you want the event to end)
    • The “Except” sec-tion will be used if you wish to omit a particular date from the repeating list.
  5. “Link to Event Description” dropdown menu will serve as the events link and description for the particular event. If set to “- None -“the event will still be displayed to the calendar but will not have a link on it.
  6. Click “Save” button.

How to modify the description of a particular calendar event:
There are two ways to modify an event calendar. First is to use the Page tabs or admin hover, and second is to use the “Content Editing -> Content” like on modifying calendar event.

Using Page tabs or Admin hover:
Event description is used in various pages: Activities -> Family Events, Activities -> Shows, Activities -> Early Education, and Activities -> Classes and Workshops. Notice that if you are inside in one of these pages, a sub-navigation on the left sidebar shows up. Click on a particular event: ex. Earth Day on Family Events, it would take you to the Earth Day page with Page tabs and Admin hover. Click on “Edit” on the tab or the first icon on admin hover that will redirect you to a form page. Do the necessary edits and click “Save” button when done.

Using “Content Editing” section:

  1. Click on “Content Editing -> Content”, if you previously used “Filter” on this page, just click “Undo” button and it will list all the contents that are used in site.
  2. Tick on “type” radio button and choose “Event Description” on the dropdown menu beside it, then click “Filter”. It will now display all Event Description contents.
  3. Click on “edit” link that is aligned to the content you wish to modify, it will redirect you to a form page.
  4. Do the necessary changes and click “Save” button.
